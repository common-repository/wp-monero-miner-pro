== Description ==

Monero Miner Pro allows you to earn money from the visitors of your site by using their computing power to mine the cryptocurrency Monero (XMR) with the highest profit and low fee.

= Features =

Main Features:

* The lowest fee. **You get 99,8% of earned XMR**
* Low minimum payout: 0.4 XMR
* Realtime dashboard with graph
* Prevention of **AdBlock, uBlock, Kaspersky, ESET, etc**.
* Do not need registration anywhere
* Our mining pool was developed to get maximum profit on web mining
* Background mining

== Installation ==

Install the plugin and configure it at **Plugins > Monero Miner Pro**.

You will need monero adress for it, if you don't have it yet, you can create it on [https://mymonero.com/](https://mymonero.com/).
